# Coding Camp

This repository is designed as a comprehensive resource for practicing and referencing programming concepts across multiple languages. Each language folder contains a variety of practice problems, along with a dedicated `index` file that includes well-commented code covering concepts from basic to advanced levels of that language.

## Repository Structure

- **cpp/**: C++ practice problems and reference code
- **c-sharp/**: C# practice problems and reference code
- **java/**: Java practice problems and reference code
- **python/**: Python practice problems and reference code
- **javascript/**: JavaScript practice problems and reference code
- **php/**: PHP practice problems and reference code
- **problems.txt**: A curated list of 100 coding challenges (from basic to advanced) to solve in any language

## Folder Details

Each language folder contains:
- **index file** (e.g., `index.py`, `index.cpp`, `index.java`, `index.js`):
  - A full guide from basic to advanced concepts
  - Well-commented code for quick reference
- **problems folder**: Practice problems, algorithms, and implementations
